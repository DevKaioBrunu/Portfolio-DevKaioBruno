## Características do Portfólio

### Design Moderno & Responsivo
- Dark theme elegante com paleta de cores pensada
- Completamente responsivo — funciona perfeitamente em mobile, tablet e desktop
- Animações suaves e efeitos visuais que não comprometem a performance

### Recursos Técnicos
- **Animação de partículas** no hero section usando Canvas
- **Scroll reveal** com Intersection Observer para efeito de entrada dos elementos
- **Carousel interativo** para projetos com navegação por dots e swipe
- **Menu hamburger** otimizado para mobile
- **Acessibilidade WAI-ARIA** — semântica HTML completa

### Performance
- Desenvolvido com **vanilla JavaScript** (sem dependências externas)
- Otimizado para carregamento rápido
- CSS puro com design tokens para fácil manutenção
- Hospedado na Vercel com otimizações automáticas
